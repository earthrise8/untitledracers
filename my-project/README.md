# H2GP Competition Project

## Overview
This project is designed to provide a dynamic website for the H2GP Competition, showcasing the Untitled Racers team. It serves as an entry point for users to learn about the competition, view team information, and access related media.

## Project Structure
```
my-project
├── src
│   ├── main.ts          # Main entry point for the application
│   └── types
│       └── index.ts     # Type definitions for data structures
├── package.json         # NPM configuration and dependencies
├── tsconfig.json        # TypeScript configuration
└── README.md            # Project documentation
```

## Getting Started

### Prerequisites
- Node.js (version 14 or higher)
- npm (Node Package Manager)

### Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/my-project.git
   ```
2. Navigate to the project directory:
   ```
   cd my-project
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Building the Project
To compile the TypeScript files, run:
```
npm run build
```

### Running the Project
To start the development server, use:
```
npm start
```
This will launch the application in your default web browser.

### Deployment
To deploy the project to GitHub Pages, follow these steps:
1. Build the project using the build command.
2. Push the `dist` folder to the `gh-pages` branch of your repository.
